# ☁️ Comfort in the Cloud – S3 Static Website Project

## 🌟 Overview

This project is part of my ongoing journey into cloud engineering, where I’m intentionally building real, hands-on experience while documenting what I learn along the way.

The goal of this project is to design, build, and deploy a **personal branded static website** that represents *Comfort in the Cloud*—a space where I showcase my growth, projects, and learning in AWS.

This is not just a one-time build.
This is a **living project** that will evolve as I continue to learn and apply new cloud concepts.

---

## 🎯 Project Goals

* Create a clean, modern, and visually engaging static website
* Build and style the site using HTML and CSS
* Deploy the site using AWS S3 static website hosting
* Document every step for learning, reflection, and sharing
* Expand the project over time with cloud-native enhancements

---

## 🛠️ What I’ve Completed So Far

### ✨ Website Design & Build

* Designed a branded “Comfort in the Cloud” homepage
* Created a hero-style layout with a custom visual identity
* Implemented navigation links for:

  * Medium (Articles)
  * LinkedIn (Connect)
* Built interactive elements with hover effects
* Positioned elements precisely using CSS (flexbox and absolute positioning)
* Tested the site locally to ensure layout and functionality

### 🧠 Key Concepts Practiced

* HTML page structure
* CSS layout and positioning
* Flexbox alignment
* Layering and visual hierarchy
* Debugging layout behavior (fixed width vs browser width)
